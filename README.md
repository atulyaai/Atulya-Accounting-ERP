# Atulya Accounting ERP

Free local-first ERP for Indian businesses with accounting, inventory, billing, invoicing, and reporting.

## Features

- Ledger, trial balance, P&L, balance sheet
- Inventory management with batch tracking
- Invoice & quotation generation (PDF/Excel)
- Purchase order management
- Payment tracking & reconciliation
- GST-compatible tax calculations
- Export to Tally, Excel, PDF

## Quick Start

```bash
pip install atulya-accounting-erp
atulya-erp init
atulya-erp serve
```

## Modules

| Module | Description |
|--------|-------------|
| Accounts | Full double-entry accounting |
| Inventory | Stock, batch, serial tracking |
| Billing | Invoices, quotes, credit notes |
| Purchases | PO, GRN, vendor management |
| Reports | P&L, balance sheet, GST reports |
